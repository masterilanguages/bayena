# Bayena Ventures

Public landing site for **Bayena Ventures** — private capital, real assets, and strategic opportunities.

Matches the full landing page from the Bayena admin app, including:

- Hero & navigation
- What We Do
- Who It's For (investors, borrowers, business owners)
- Investment Opportunities
- Why Bayena Ventures
- How It Works
- Who We Work With
- Contact form
- Legal disclaimer

## Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — same content as `/bayena-ventures` in the admin app.

## Deploy

```bash
npm run build
npm start
```
