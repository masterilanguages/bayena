# Bayena

Public landing page for **Bayena** — branding and growth studio.

## Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

```bash
npm run build
npm start
```
